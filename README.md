# sheild_app

SHEild is a Flutter mobile app paired with a smart ESP32 wristband 
that uses AI to predict danger and monitor women's health in real time 
— not just react to it.

1. SOS Trigger      — Triple-tap the band or tap in-app to send SMS alerts + live GPS to guardians  
2. AI Risk Detection — LSTM model watches heart rate, motion, GPS & sound to alert before danger  
3. Health Dashboard  — Tracks menstrual cycle, stress trends & heart rate variability  
4. Guardian Network  — Trusted contacts see live location & get emergency notifications  
5. Live Map          — Real-time location, risk zones & safer route suggestions

Built with: Flutter • Firebase • Node.js • TensorFlow Lite • Google Maps SDK • Twilio SMS • Bluetooth BLE
