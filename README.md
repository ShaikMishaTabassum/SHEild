# SHEild

SHEild is a Flutter mobile app paired with a smart ESP32 wristband 
that uses AI to predict danger and monitor women's health in real time 
— not just react to it.

🔴 SOS Trigger      — Triple-tap the band or tap in-app to send SMS alerts + live GPS to guardians  
🤖 AI Risk Detection — LSTM model watches heart rate, motion, GPS & sound to alert before danger  
💚 Health Dashboard  — Tracks menstrual cycle, stress trends & heart rate variability  
👥 Guardian Network  — Trusted contacts see live location & get emergency notifications  
🗺️ Live Map          — Real-time location, risk zones & safer route suggestions

Built with: Flutter • Firebase • Node.js • TensorFlow Lite • Google Maps SDK • Twilio SMS • Bluetooth BLE
